## 🚀 App Store Connect Setup

### Step 1: Create App Listing

1. Log in to App Store Connect
2. Click "My Apps" → "+" → "New App"
3. Fill in basic information:
    - Platform: iOS
    - Name: Hex Zero
    - Primary Language: English (U.S.)
    - Bundle ID: com.hexzero.game
    - SKU: hexzero-ios

### Step 2: App Information

1. Navigate to App Information
2. Set category: Games > Puzzle
3. Add secondary category: Games > Strategy
4. Set content rights (if applicable)
5. Set age rating: 4+

### Step 3: Pricing and Availability

1. Set price: Free
2. Set availability: All countries
3. Set release date: Manual release

### Step 4: App Store Listing

1. Upload screenshots for all required device sizes
2. Add app description and promotional text
3. Set keywords
4. Add support and marketing URLs
5. Upload app icon (1024×1024)

### Step 5: App Review Information

1. Add app review notes
2. Confirm demo account not needed
3. Add contact information

## 📊 Build and Upload Process

### Step 1: Prepare Release Build

```bash
# Build web assets
npm run build

# Sync to iOS
npx cap sync ios

# Open in Xcode
npx cap open ios
```

### Step 2: Archive in Xcode

1. Select "Generic iOS Device" or connected device
2. Product → Archive
3. Wait for archive to complete
4. Distribute App → App Store Connect
5. Upload to App Store Connect

### Step 3: Processing and Review

1. Wait for processing (15-30 minutes)
2. Add build to app listing
3. Submit for App Review
4. Review typically takes 24-48 hours

## 🔧 Troubleshooting

### Common Issues

1. **Build Upload Fails**

    - Check certificate and provisioning profile
    - Verify bundle ID matches exactly
    - Ensure version number is incremented

2. **App Rejected**

    - Address reviewer feedback
    - Update metadata if needed
    - Resubmit with fixes

3. **Screenshots Rejected**
    - Ensure actual app content is shown
    - Remove any placeholder text
    - Follow screenshot guidelines exactly
