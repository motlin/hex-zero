# 🍎 App Store Submission Materials

This directory contains all materials needed for submitting Hex Zero to the iOS App Store.

## 📁 Directory Contents

### Documentation

-   **`APP_STORE_SUBMISSION.md`** - Complete submission guide and checklist
-   **`APP_STORE_COPY.md`** - All text content and metadata for the listing
-   **`TESTFLIGHT_SETUP.md`** - Beta testing setup guide

### Generated Assets

-   **`screenshots/`** - App Store screenshots (generated by script)
-   **`metadata/`** - Additional metadata files (if needed)

## 🚀 Quick Start

### 1. Generate Screenshots

```bash
npm run screenshots
```

### 2. Review App Store Copy

Edit `APP_STORE_COPY.md` to customize:

-   App description
-   Keywords
-   Screenshot captions
-   Support URLs

### 3. Follow Submission Guide

Use `APP_STORE_SUBMISSION.md` for step-by-step submission process.

### 4. Optional: Set Up TestFlight

Use `TESTFLIGHT_SETUP.md` for beta testing before public release.

## 📋 Submission Checklist

-   [ ] Screenshots generated for all required device sizes
-   [ ] App description finalized and optimized
-   [ ] Keywords researched and selected
-   [ ] Privacy policy accessible online
-   [ ] Support and marketing URLs ready
-   [ ] Distribution certificate and provisioning profile configured
-   [ ] Archive build uploaded to App Store Connect
-   [ ] Age rating and content descriptions completed

### iPhone

-   6.7" Display (iPhone 15 Pro Max) - **Required**
-   6.1" Display (iPhone 15) - **Required**

-   12.9" iPad Pro - **Required**
