# Hooks

This directory contains custom React hooks for the Hex Zero React Native app.

## Hook Types

- **Game State Hooks**: Hooks for accessing and managing game state
- **Animation Hooks**: Custom hooks for complex animations
- **Gesture Hooks**: Hooks for handling touch gestures and interactions
- **Storage Hooks**: Hooks for persistent storage operations
- **Navigation Hooks**: Custom navigation logic and state

## Naming Convention

- Use camelCase with "use" prefix (e.g., `useGameState.ts`)
- Each hook should have a single, clear purpose
- Export hooks as named exports
- Include TypeScript types for hook parameters and return values
