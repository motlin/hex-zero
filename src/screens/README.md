# Screens

This directory contains full-screen React Native components that represent different screens/pages in the Hex Zero app.

## Screen Types

- **GameScreen**: Main game playing screen with the hex board and controls
- **MenuScreen**: Main menu with game options and navigation
- **DifficultyScreen**: Screen for selecting game difficulty
- **SettingsScreen**: App settings and preferences
- **HelpScreen**: How to play and game instructions
- **StatsScreen**: Game statistics and achievements

## Naming Convention

- Use PascalCase with "Screen" suffix (e.g., `GameScreen.tsx`)
- Each screen should be a complete, navigable view
- Screens can compose multiple components from the components directory
